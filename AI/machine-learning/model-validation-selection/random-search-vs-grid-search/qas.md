# Random Search vs Grid Search - Q&A

---

## Core Idea

**Q: When is random search often better?**
A: When only some hyperparameters matter and budget is limited.

**Q: What controls random search cost?**
A: The number of sampled trials.

**Q: What distribution is common for learning rate?**
A: Log-uniform.

## Comparison

**Q: When is grid search fine?**
A: Small search spaces with known good candidate values.
