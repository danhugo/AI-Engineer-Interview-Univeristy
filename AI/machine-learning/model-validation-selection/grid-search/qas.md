# Grid Search - Q&A

---

## Core Idea

**Q: What does grid search do?**
A: Evaluates every hyperparameter combination in a predefined grid.

**Q: What is its main benefit?**
A: Simple and exhaustive within the grid.

**Q: What is its main weakness?**
A: Combinations grow quickly with more parameters.

## Validation

**Q: Should the test set guide grid search?**
A: No. Use validation/CV, then test once at the end.
