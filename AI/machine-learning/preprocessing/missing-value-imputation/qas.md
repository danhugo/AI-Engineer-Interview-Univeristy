# Missing Value Imputation - Q&A

---

## Core Idea

**Q: What is imputation?**
A: Filling missing values using learned replacement values.

**Q: What strategies are common?**
A: Mean, median, and mode.

**Q: Where should replacement values be learned?**
A: Training data only.

## Signal

**Q: When add a missing indicator?**
A: When the fact that a value is missing may be predictive.
