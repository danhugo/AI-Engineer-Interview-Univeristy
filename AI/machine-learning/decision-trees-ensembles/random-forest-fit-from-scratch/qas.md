# Random Forest Fit from Scratch - Q&A

---

## Core Idea

**Q: What is a random forest?**
A: Bagged decision trees with feature randomness.

**Q: Why bootstrap rows?**
A: To make different trees see different data.

**Q: Why random features?**
A: To decorrelate trees.

## Prediction

**Q: How does classification prediction work?**
A: Majority vote or averaged probabilities.

**Q: What does averaging reduce?**
A: Variance.
