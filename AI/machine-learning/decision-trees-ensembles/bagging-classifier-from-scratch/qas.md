# Bagging Classifier from Scratch - Q&A

---

## Core Idea

**Q: What does bagging mean?**
A: Bootstrap aggregating.

**Q: What does it reduce?**
A: Variance.

**Q: How does a classifier aggregate?**
A: Majority vote or averaged probabilities.

## Comparison

**Q: How is random forest different?**
A: It adds random feature selection to bagged trees.
