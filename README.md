# AI Engineer and Software Engineer

Now both roles seem to be converging, but there's still a seperate area of knowledge needed for AI Engineer role. Let's say a software engineer works with deterministic problems while an AI engineer works with probabilistic problems. Beyond just building, and integrating AI system, AI engineers need to build a regulated, reliable, cost-bounded production workflow. To do that, having fundamental knowledge in AI/ML is essential.
